Gekozen Installatiemethode: NPM Lokaal

Waarom NPM Lokaal?

Ik heb gekozen voor de NPM (Node Package Manager) lokale installatie omdat:

1: Offline kan testen of alles er goed uit ziet. 2: Ik kan precies kiezen welke versie ik wil gebruiken. 3: Als ik zou willen zou ik de Bootstrap bestanden kunnen aanpassen. 4: Het heeft vaak een snellere laadtijd omdat het van dezelfde server komt. 5: Het is makkelijk te beheren (naar mijn mening)

Installatiestappen NPM Methode

Stap 1: NPM Project Initialiseren

Open terminal en voer uit:

npm init -y

Stap 2: Bootstrap Installeren

Installeer Bootstrap via NPM:

npm install bootstrap

Stap 3: Bootstrap CSS Linken

Verwijs naar het lokale Bootstrap CSS bestand:

<link href="node_modules/bootstrap/dist/css/bootstrap.min.css" rel="stylesheet">

Stap 4: Bootstrap JavaScript Toevoegen

Voeg het lokale JavaScript bestand toe:

<script src="node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
