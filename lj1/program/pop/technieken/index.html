De technieken

Op deze pagina ga ik uitleggen welke technieken ik heb gebruikt om deze website te maken.

CSS

Keyframes

Om de titels te van kleur te laten veranderen heb ik gebruik gemaakt van keyframes en een mooie linear gradient. De keyframes zorgen ervoor dat de kleur van de tekst veranderd door de achtergrond te bewegen. Technisch gezien is zie je namelijk de achtergrond en niet de tekst zelf. 

                    
                        .fancy {
                            background: rgb(0, 43, 255);
                            background: linear-gradient(90deg, rgba(0, 43, 255, 1) 2%, rgba(255, 0, 0, 1) 26%, rgba(255, 212, 0, 1) 50%, rgba(0, 255, 5, 1) 74%, rgba(0, 43, 255, 1) 98%);
                            background-clip: text;
                            -webkit-background-clip: text;
                            -webkit-text-fill-color: transparent;
                            animation: fancyAnimation 7s linear infinite alternate;
                            background-size: 200% 100%;
                        }

                        @keyframes fancyAnimation {
                        0% {
                                background-position: 0% 0%;
                            }

                            100% {
                                background-position: 200% 0%;
                            }
                        }

JavaScript

Math.random()

Math.random() genereert een getal tussen de 0 en 1. Dit getal kan je voor heel veel verschillende dingen gebruiken. Zoals ik heb gedaan het selecteren van een willekeurig item uit een string om junk data te genereren. 

                    
                        function generateData() {
                            let data = '';
                            let possible = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789!@#$%^&*()_+-={}[]\|:;,.?/~`'";
                            let custom_dict = document.getElementById('dictionary').value;
                            let num_of_data_strings = parseInt(document.getElementById('num_of_strings').value);
                            let data_length = parseInt(document.getElementById('length').value);
                            let separator = document.getElementById('separator').value;
                        
                            if (custom_dict.length > 0) {
                                possible = custom_dict.split('').filter((item, pos, self) => self.indexOf(item) === pos).join('');
                            }
                        
                            for (let i = 0; i < num_of_data_strings; i++) {
                                for (let i = 0; i < data_length; i++) {
                                    data += possible.charAt(Math.floor(Math.random() * possible.length));
                                }
                                data += separator;
                            }
                            document.getElementById('result').innerHTML = data;
                        }

Dag deel bepalen

Om te bepalen of het ochtend, middag, avond of nacht is heb ik een functie geschreven die de huidige tijd pakt, en dan kijkt welk uur het is en dan afhankelijk daarvan word er goedemorgen, goedemiddag, goedenavond of goedenacht gezegd. 

                    
                        const hours = new Date().getHours();
                        let greet = '';
                        if (hours > 0 && hours < 6) {
                            greet = 'Goedenacht';
                        }
                        else if (hours >= 6 && hours < 12) {
                            greet = 'Goedemorgen';
                        }
                        else if (hours >= 12 && hours < 18) {
                            greet = 'Goedemiddag';
                        }
                        else if (hours >= 18 && hours < 24) {
                            greet = 'Goedenavond';
                        }
                        document.addEventListener('DOMContentLoaded', function() { 
                            if (greet === '') {
                                stop;
                                console.error('[error] Could not determine time of day.');
                            }
                            document.getElementById('greet').innerHTML = `${greet}, beste lezer!`;
                        });
