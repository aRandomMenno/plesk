Mijn interesses en doelen

Ik houd heel erg van programmeren, cyber-security, computers, gamen en fotograferen. Een paar dingen die ik graag wil leren buiten program 1 zijn:

  • Andere programmeertalen leren zoals Rust, Python en C/C++
  • Leren hoe ciphers als caesar cipher, vignere cipher etc werken
  • Een begin maken aan ethisch hacken
  • Mijn eigen website maken met (backend en) een mooie domein naam

Waarom en waarvoor?

Ik wil graag andere programmeertalen leren omdat ik dan meer mogelijkheden heb om dingen te maken. De reden dat ik graag wil leren hoe ciphers werken is omdat ik het interessant vind om hoe mensen vroeger geheime boodschappen verstuurden en ik het grappig vind, je kan het als een soort geheim taal gebruiken. De reden dat ik wil leren ethisch hacken is omdat ik het leuk vind om te leren hoe je dingen kan beveiligen en hoe je dingen kan hacken.

De grootste reden om mijn eigen website te maken is omdat ik het leuk vind om met software en computers te sleutelen. Ik wil graag leren hoe ik een website kan hosten en hoe ik een een backend kan maken. En een mooi domein naam zou wel fijn zijn. Momenteel gebruik ik namelijk GitHub pages.

Hoe ga ik dit leren?

De meeste dingen op mijn lijstje kan ik leren door online leermiddelen op te zoeken, op YouTube tutorials te kijken op websites voorbeelde bekijken en door te experimenteren en oefenen. Voor ethisch hacken kan ik volgend jaar meedoen met de Hack 010, dan mag je proberen om de Rotterdamse gemeente te hacken. En ik kan meedoen aan CTF's (Capture The Flag) challenges.

Om meer te leren over Ciphers heb ik een website gevonden waar je heel veel verschillende hebt, dcode.fr. Maar ik kan ook andere websites zoeken.

Hoe ik mijn eigen website ga maken met eigen domain weet ik nog niet precies, maar ik kan wel onderzoek doen naar hoe ik een domain kan kopen en een VPS kan "huren". Misschien kan ik ook een backend toevoegen.

Om andere programmeertalen te leren kan ik op YouTube tutorials kijken, Blogposts lezen, andere mensen vragen en zelf experimenteren. Ik kan ook kijken of er een cursus is die ik kan volgen.
