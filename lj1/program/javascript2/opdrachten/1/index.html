Wanneer gebruik je een for of while loop?

For Loop

Een for loop wordt gebruikt wanneer je een stuk code een vast aantal keren wilt herhalen. Hier zijn twee situaties waarin je een for loop zou gebruiken:

  1. Itereren over een array: Wanneer je door alle elementen van een array wilt lopen en een bewerking op elk element wilt uitvoeren.
  2. Herhalen van een actie: Wanneer je een actie een specifiek aantal keren wilt uitvoeren, zoals het genereren van een reeks getallen.

While Loop

Een while loop wordt gebruikt wanneer je een stuk code wilt herhalen zolang een bepaalde conditie waar is. Hier zijn twee situaties waarin je een while loop zou gebruiken:

  1. Wachten op een gebeurtenis: Wanneer je wilt blijven wachten tot een bepaalde gebeurtenis plaatsvindt, zoals het wachten op gebruikersinvoer.
  2. Itereren tot een conditie verandert: Wanneer je door een dataset wilt lopen totdat een bepaalde conditie verandert, zoals het doorlopen van een lijst totdat een specifiek element wordt gevonden.
